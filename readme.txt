Create simple to complex Buy One Get One free discount pricing deals in your  WooCommerce online store.

== Description ==
Buy one get one free (BOGO) is a most popular discount type offered by online stores. The plugin helps you create these BOGO discount deals easily.
The plugin currently allows you to create Buy X Get X type of deals. That is when a customer purchases a product, he can get another quantity of the same product free.
The plugin offers a repeat option, which means you do not have to configure the ranges.

= Key Features =

* Create Buy X and Get X deals
* Repeat option so that you dont have to specify ranges
* Product specific
* Category based. You can select a category and offer the deal to all the products in that category
* Easy to understand interface

== Installation ==
1. Upload the plugin files to the /wp-content/plugins/  directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the Plugins screen in WordPress
3. Once plugin is activated, click on the Settings on the Plugin page under the WooCommerce BOGO Rules plugin or Click on the WooCommerce BOGO Deals menu on the wordpress admin page.
4. Create and manage various BOGO rules at ease.

== Frequently asked questions ==

= Are multiple discounts possible ? What is the limit for the number of rules ? =

Yes. It is possible to create multiple discount and you can setup any number of discount rules in your store.
